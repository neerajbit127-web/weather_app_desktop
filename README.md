# 🌦️ Weather App

A simple and elegant Python Weather App that provides real-time weather information for any city using a weather API.

---

## ✨ Features

- 🌍 Search weather by city
- 🌡️ Current temperature
- ☁️ Weather description
- ⚡ Fast and lightweight

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/YourUsername/weather-app.git
```

Go into the project

```bash
cd weather-app
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run

```bash
python main.py
```

---

## 🔑 API Key

Create a `.env` file

```
API_KEY=YOUR_API_KEY
```
